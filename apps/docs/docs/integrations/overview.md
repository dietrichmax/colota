# Choosing a Backend

Colota sends location data to a server you control. Choose from the following backend options:

## Available Backends

- [Dawarich](dawarich.md)
- [GeoPulse](geopulse.md)
- [Overland](overland.md) - any Overland-compatible server
- [Home Assistant](home-assistant.md)
- [OwnTracks](owntracks.md)
- [Reitti](reitti.md)
- [PhoneTrack](phonetrack.md)
- [Traccar](traccar.md)
- [Custom Backend](custom-backend.md)

## No backend needed

Colota works fully offline without any server. Locations are stored on-device and can be exported as CSV, GeoJSON, GPX, or KML at any time.
